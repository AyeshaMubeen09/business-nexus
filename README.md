# Business Nexus – Investor & Entrepreneur Collaboration Platform

A full-stack collaboration platform built to connect **Entrepreneurs** and **Investors** through secure authentication, real-time communication, meeting scheduling, and document management.

## 🚀 Features

* 🔐 JWT Authentication & Role-Based Authorization
* 👤 User Registration, Login & Profile Management
* 📅 Meeting Scheduling with Conflict Detection
* 📹 Real-Time Video Calling (WebRTC + Socket.IO)
* 📄 Document Chamber

  * Upload PDF Documents
  * Advanced PDF Preview
  * Continuous Scrolling
  * Zoom Controls
  * Fullscreen Viewer
  * Secure Download
  * Delete Documents
  * Electronic Signature Support
* 🎨 Responsive Modern UI built with React & Tailwind CSS
* 🔒 Secure REST APIs with MongoDB Persistence
